# Art & UI Style Guide (Diablo 2 vibe)

> Note: I couldn't pull the exact prompt set from the referenced chat ID, so this is a high-quality reconstruction.
> If you paste the original prompts later, we can replace this file 1:1.

## Pillars
- Dark gothic fantasy (Diablo 2 inspired): stone/iron/leather, worn textures.
- Readability first: strong silhouettes, limited saturation, clear rarity cues.
- UI is DOM/CSS; **avoid text baked into images** (text is rendered by UI).

## Universal prompt add-on
Use this suffix in every prompt:

**"dark gothic fantasy UI, Diablo 2 inspired, worn stone and iron frame, subtle parchment texture, high contrast readable, minimal glow"**

## Negative prompt (always)
**"modern flat UI, neon cyberpunk, anime, glossy 3D plastic, photoreal faces, watermark, signature, text, logos"**

## UI components prompts
### Inventory panel
"inventory panel UI, carved stone border, iron rivets, dark parchment background, subtle grime, medieval gothic, empty slots grid, no text"

### Item tooltip frame
"item tooltip frame, ornate gothic border, parchment interior, subtle noise, high readability, no text"

### Buttons
"gothic game button, stone and iron, bevelled edges, subtle hover glow, icon-only, no text"

## Item icons prompts
"isometric item icon, hand-painted, Diablo 2 style, muted palette, strong silhouette, clean edges, transparent background"

## Export rules
- Prefer PNG with transparency for icons/frames.
- Provide 2 sizes when possible (1x and 2x).
- No letters/words inside images.
