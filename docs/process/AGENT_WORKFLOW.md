# Process — как агенту работать со стилем (v0.1)

## Перед любой генерацией

1) Открыть `docs/style/STYLE_BIBLE.md`
2) Взять нужный шаблон из `docs/style/PROMPTBOOK.md`
3) Проверить размер/сетку в `docs/style/ASSET_SPECS.md`
4) Генерировать **сразу наборами** (состояния/серии), а не по одному файлу.

## Ревью циклами (быстро)

- Цикл 1: форма/силуэт/читаемость (без микродеталей)
- Цикл 2: материалы/свет
- Цикл 3: текстуры/пыль (слабее!)
- Цикл 4: финальная консистентность серии

## Как формулировать фидбек

Плохой: “некрасиво”  
Хороший:
- “слишком плоско → добавь depth: inner shadow + specular highlight на металле”
- “шумно → уменьшить текстуру на 30% и повысить контраст текста”
- “не та семья → сравнять углы/накладки/толщину рамки”

## Definition of Done (UI/ассет)

- Вписывается в Style Bible
- Есть все состояния (если компонент интерактивный)
- Легко верстается (9-slice / прозрачность / размеры)
- Серия выглядит как один набор
