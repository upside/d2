# UI Components — контракты (v0.1)

> Цель: агент **собирает интерфейс из компонентов**, а не “рисует страницу целиком”.

## Общие правила

- Все компоненты должны быть **состояниями одной семьи**: default / hover / pressed / disabled / selected (если применимо).
- Везде одинаковые: радиусы/скосы, металл, уровень шума, тени.
- Размеры и отступы фиксируются в пикселях.

## 1) Panel / Frame (большие панели)

**Назначение:** фон для блоков: персонаж, инвентарь, чат, бой.

**Спеки:**
- Минимальная толщина рамки: 14–22px (в зависимости от масштаба).
- Углы: “металлические накладки” + скос.
- Внутренний padding контента: 16–24px.
- **9‑slice friendly:** углы отдельные, борта повторяемые.

**Состояния:** обычно нет, но может быть “активная панель” (легкий внутренний glow).

## 2) Button (кнопки)

**Варианты:** primary / secondary / danger (редко).

**Размеры:**
- SM: 32px высота, padding 12x8
- MD: 40px высота, padding 16x10
- LG: 48px высота, padding 20x12

**Состояния:**
- default: спокойный металл/камень
- hover: чуть ярче спекуляр + лёгкий inner highlight
- pressed: “вдавленность” (inner shadow сильнее)
- disabled: контраст ниже, текст приглушён

## 3) Inventory Slot

**Размер:** базово 64x64 (можно 72/80 для крупных).

**Слои:**
1) каменный/металлический “корпус”
2) внутренний “колодец” (темнее)
3) лёгкая внутренняя тень (depth)
4) подсветка hover: тонкое кольцо + мягкий bloom

**Состояния:** empty / filled / hover / selected / locked.

## 4) Socket (ячейка сокета)

**Размер:** 22–28px (зависит от UI масштаба).

**Слои:**
- “гнездо” с глубиной + окантовка
- индикатор locked: маленький штифт/замок (без текста)
- вставка (rune/gem): центрируется, одинаковые отступы.

## 5) Tooltip (предмет/умение)

**Ширина:** 360–440px (авто по контенту допустим, но с max‑width).

**Структура:**
- Header: имя предмета + линия/разделитель
- Core: базовые характеристики
- Mods: аффиксы/суффиксы
- Footer: требования/уровень/сеты

**Правила читаемости:**
- Контраст текста ≥ AA (на глаз: белый/пергамент на почти чёрном).
- Межстрочный 1.25–1.35.
- Разделители тонкие, не кричащие.

## 6) Icon Badge (редкость/статус)

- Размер: 14–18px
- Использовать как “точка” рядом с названием, не как гигантский баннер.

## 7) Divider / Ornament

- Горизонтальные линии: 1–2px + лёгкая текстура.
- Орнаменты — только в ключевых местах, не везде.

## 8) Макеты страниц (как собирать)

### Инвентарь/персонаж
- Левая панель: персонаж/экипировка
- Центр: инвентарь + сокеты
- Правая: тултип/описание/чат

### Бой
- Центр: лог событий (текст)
- Слева/справа: участники/статы
- Внизу: панель действий (кнопки/умения)

## 9) Чек‑лист приёмки UI

- [ ] Все состояния компонентов выглядят как “одна семья”
- [ ] Текст читается на 1366x768 и 1920x1080
- [ ] Hover не “неонит”, а подчёркивает материал
- [ ] Текстуры не спорят с иконками/текстом
